Estetik Beauty Studio Website

Modern responsive beauty studio website built with React, Vite, Tailwind CSS and Firebase Authentication.

Features

- Responsive modern UI
- Dark / Light mode
- Firebase Authentication
- Dynamic services system
- Nested service categories
- Image gallery
- Partner brands section
- Contact page

Technologies

- React
- Vite
- Tailwind CSS
- Firebase
- React Router

Installation

bash npm install npm run dev
